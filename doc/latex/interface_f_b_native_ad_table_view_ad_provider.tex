\subsection{Detailed Description}
Additional functionality on top of \hyperlink{interface_f_b_native_ads_manager}{F\-B\-Native\-Ads\-Manager} to assist in using native ads within a U\-I\-Table\-View. This class contains a mechanism to map index\-Paths to native ads in a stable manner as well as helpers which assist in doing the math to include ads at a regular interval within a table view.
