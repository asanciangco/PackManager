Detailed Description

A dialog for creating app groups.

Method Documentation

- (BOOL) canShow

A Boolean value that indicates whether the receiver can initiate an app group dialog. May return NO if the appropriate Facebook app is not installed and is required or an access token is required but not available. This method does not validate the content on the receiver, so this can be checked before building up the content.
