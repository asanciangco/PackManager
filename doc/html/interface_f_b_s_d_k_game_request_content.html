Additional freeform data you may pass for tracking. This will be stored as part of the request objects created. The maximum length is 255 characters.

- (FBSDKGameRequestFilter) filters
readwritenonatomicassign

This controls the set of friends someone sees if a multi-friend selector is shown. It is FBSDKGameRequestNoFilter by default, meaning that all friends can be shown. If specify as FBSDKGameRequestAppUsersFilter, only friends who use the app will be shown. On the other hands, use FBSDKGameRequestAppNonUsersFilter to filter only friends who do not use the app. The parameter name is preserved to be consistent with the counter part on desktop.
