This is a method to disable auto refresh for the \hyperlink{interface_f_b_ad_view}{F\-B\-Ad\-View} instance

By default, we read the refresh interval from the placement setting in your Facebook developers page. Once you call this method, the auto refresh will be disabled for this \hyperlink{interface_f_b_ad_view}{F\-B\-Ad\-View} instance, and you cannot re-\/enable the refresh for this instance. A new created \hyperlink{interface_f_b_ad_view}{F\-B\-Ad\-View} will still use the default behavior.

This method is designed for ad network mediation.
