\subsection{Detailed Description}
Represents a pending request to navigate to an App Link. Most developers will simply use navigate\-To\-U\-R\-L\-In\-Background\-: to open a U\-R\-L, but developers can build custom requests with additional navigation and app data attached to them by creating B\-F\-App\-Link\-Navigations themselves.
