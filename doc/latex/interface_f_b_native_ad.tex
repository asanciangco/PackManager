\subsection{Detailed Description}
The \hyperlink{interface_f_b_media_view}{F\-B\-Media\-View} loads media content from a given \hyperlink{interface_f_b_native_ad}{F\-B\-Native\-Ad}. This view takes the place of manually loading a cover image.

The \hyperlink{interface_f_b_native_ad}{F\-B\-Native\-Ad} represents ad metadata to allow you to construct custom ad views. See the Native\-Ad\-Sample in the sample apps section of the Audience Network framework.
