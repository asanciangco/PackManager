\subsection{Detailed Description}
The \hyperlink{interface_f_b_s_d_k_messenger_sharer}{F\-B\-S\-D\-K\-Messenger\-Sharer} is used to share media from apps into Messenger. The underlying mechanism used to share data between apps is U\-I\-Pasteboard


\begin{DoxyItemize}
\item Facebook\-App\-I\-D must be set in the your app's Info.\-plist with the Facebook App Id
\item Any existing data in the system's public pasteboard will get overwritten with the shared media
\item Once the data is shared in Messenger, the pasteboard with be cleared
\item The following strings need to be translated in your app\-: N\-S\-Localized\-String("Get Messenger\char`\"{}, @\char`\"{}Alert title telling a user they need to install Messenger\char`\"{})
   N\-S\-Localized\-String(@\char`\"{}You'll need to install the latest version of Messenger to send this.\char`\"{}, @\char`\"{}Alert message when an old version of messenger is installed\char`\"{})
   N\-S\-Localized\-String(@\char`\"{}Not Now\char`\"{}, @\char`\"{}Button label when user doesn't want to install Messenger\char`\"{})
   N\-S\-Localized\-String(@\char`\"{}Install\char`\"{}, @\char`\"{}Button label to install Messenger\char`\"{})
   N\-S\-Localized\-String(@\char`\"{}Send\char`\"{}, @\char`\"{}Button label for sending a message") 
\end{DoxyItemize}
